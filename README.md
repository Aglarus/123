# Музыкальный Telegram Бот

Многоязычный бот для поиска, распознавания и скачивания музыки.

## Основные возможности

- **Выбор языка**: Поддержка Русского, Узбекского, Английского и Азербайджанского языков при первом запуске (`/start`).
- **Поиск музыки**: Поиск треков по названию или исполнителю через YouTube (используется `yt-dlp`).
- **Распознавание**: Узнает песню по голосовому сообщению или аудиофайлу (используется Shazam API через `shazamio`).
- **Скачивание**: Позволяет скачать выбранный трек прямо в Telegram (до 10 минут, формат mp3/m4a).

## Технологический стек

- **Python 3.11+**
- **python-telegram-bot**: Основной фреймворк для Telegram API.
- **yt-dlp**: Поиск метаданных и загрузка аудио.
- **shazamio**: Распознавание песен.
- **pydub**: Конвертация аудио (требуется ffmpeg).
- **ffmpeg**: Системная зависимость для обработки медиа.

## Установка и запуск

1. **Токен бота**: Получите токен у [@BotFather](https://t.me/BotFather) и добавьте его в секреты Replit (или переменные окружения) под именем `TELEGRAM_BOT_TOKEN`.
2. **Запуск**: Бот настроен через Replit Workflow. Нажмите кнопку **Run** для запуска.

## Использование

1. Отправьте `/start` боту.
2. Выберите предпочтительный язык интерфейса.
3. Отправьте текстовый запрос (например, "Imagine Dragons Believer") для поиска.
4. Или отправьте голосовое/аудио сообщение для распознавания музыки.

## Локализация

Все системные сообщения и кнопки переведены на 4 языка и хранятся в словаре `STRINGS` в файле `main.py`. Выбор пользователя сохраняется в оперативной памяти (`user_prefs`).
